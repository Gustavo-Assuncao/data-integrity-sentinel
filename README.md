# Sentinela de Integridade de Dados 🛡️

Framework em **Python (POO)** projetado para auditoria e saneamento de ecossistemas de **Big Data** (68M+ registros).

## 🚀 Arquitetura Técnica
Este motor foi construído sob o princípio de **escalabilidade horizontal**, utilizando processamento em *chunks* para gerenciar volumetrias massivas sem degradação de performance de hardware.

## 🛠️ Stack de Engenharia
- **Python 3.10+**: Core do processamento.
- **Pandas**: Motor de manipulação de vetores de dados.
- **Regex (E.164)**: Normalização técnica para conformidade internacional.

## 📊 Aplicação do Case
Auditando 68 milhões de registros para garantir a integridade de pipelines de Growth e IA.
